Order Management System

Simple order management system written in Java.  
The project simulates handling clients, products, orders, and payments.

Features
- Create and manage clients
- Add products (physical and fragile)
- Create orders with multiple items
- Handle payments and payment status
- Manage order status

Technologies
- Java
- Object-Oriented Programming (OOP)

 Project Structure
- `model` – data classes
- `repository` – data storage logic
- `service` – business logic

How to run
1. Clone the repository
2. Open the project in IntelliJ IDEA
3. Run

This project was created to practice object-oriented programming, working with structured code, and basic system design.
